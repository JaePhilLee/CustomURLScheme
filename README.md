# CustomURLScheme
Run PC Program in Chrome Browser

# 환경 구성
1. VLC 설치 (*설치 경로 확인 필수)
  > 기본 경로 : C:\Program Files\VideoLAN\VLC

1-2. [1번에서 경로가 다를 경우에만 진행]
  > Windows 메모장 실행
  > rtsp.reg 파일 Drag&Drop
  > 최하단 부 경로 수정 [유의 사항 : \ => \\]

2. rtsp.reg 파일 실행

3. html 코드 작성
  > 예제 파일의 <a href="rtsp://{rtsp url 경로}"></a> 부분 경로 수정
